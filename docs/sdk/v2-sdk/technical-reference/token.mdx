# Token

```
constructor(chainId: ChainId, address: string, decimals: number, symbol?: string, name?: string)
```

## Example
```js
import { ChainId, Token } from "@koindx/v2-sdk";
const token = new Token(ChainId.MAINNET, '17n12ktwN79sR6ia9DDgCfmw77EgpbTyBi', 18, 'VHP', 'Virtual Hash Power')
```
